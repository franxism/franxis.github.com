1. INTRODUCCIÓN
---------------

PCSX4ALL es un emulador de PSOne (PSX) para la consolas portátiles WIZ/CAANOO de GPH.
Se trata de un port del emulador PCSX Reloaded.
Para ver la licencia GPL mira el final de este documento (capítulo 14).

Las páginas oficiales de este proyecto (con noticias, información adicional y donaciones) son:
http://chui.dcemu.co.uk/
http://franxis.zxq.net/


2. CONTROLES
------------

Los controles son los siguientes:
- Joystick: Movimiento del pad.
- B,X,A,Y,L,R,SELECT+L ó I+L,SELECT+R ó I+R: Botones Círculo,X,Cuadrado,Triángulo,L,R,L2,R2.
- SELECT ó I: Botón SELECT.
- MENU ó II: Botón START.
- L+R+START ó HOME: Salir al menú de selección de juegos (después L+R ó HOME en el menú para salir).
- L+R+SELECT ó L+R+I: Mostrar FPS.
- VolUp and VolDown: Incrementar / decrementar el volumen.
- SELECT+VolDown ó I+L: Cargar save-state.
- SELECT+VolUp ó I+R: Salvar save-state.


3. OPCIONES DEL EMULADOR
------------------------

- WIZ/CAANOO Clock:
Se puede seleccionar frecuencias de 200 a 900 MHz. El rendimiento del emulador es mejor a frecuencias más altas.
533 MHz es la frecuencia estándar. 700 MHz parece funcionar ok con cualquier WIZ (pero la duración de las pilas se reduce).
¡Usa frecuencias más altas bajo tu propia responsabilidad!

- RAM Tweaks:
ON: Se activan los tweaks de la RAM para ganar más rendimiento. Algunas consolas presentan problemas con esta opción. Recomendado en WIZ.
OFF: Se desactivan los tweaks de la RAM para asegurar el funcionamiento en cualquier consola. Recomendado en CAANOO.

- Frame-Limit:
ON: Limitador de frames habilitado.
OFF: Limitador de frames deshabilitado.

- Frame-Skip:
Frame-Skip OFF: Frame-skip deshabilitado
Frame-Skip ON (if spd<50%): Frame-skip automático si velocidad < 50%
Frame-Skip ON (if spd<60%): Frame-skip automático si velocidad < 60%
Frame-Skip ON (if spd<70%): Frame-skip automático si velocidad < 70%
Frame-Skip ON (if spd<80%): Frame-skip automático si velocidad < 80%
Frame-Skip ON (if spd<90%): Frame-skip automático si velocidad < 90%
Frame-Skip ON (minimum): Frame-skip habilitado (mínimo)
Frame-Skip ON (medium): Frame-skip habilitado (medio)
Frame-Skip ON (maximum): Frame-skip habilitado (máximo)

- Interlace:
OFF: Entrelazado de video desactivado.
Simple: Entrelazado simple (sólo se muestran la mitad de las líneas).
Progressive: Entrelazado progresivo (las líneas pares e impares se muestran alternativamente).

- Sound:
OFF: Sonido desactivado.
ON (basic): Sonido habilitado (sólo FM y WAVE).
ON (XA): Sonido extra habilitado (XA-Audio).
ON (CD-Audio): Sonido extra habilitado (CD-Audio).
ON (XA+CD-Audio): Sonido extra habilitado (XA-Audio y CD-Audio).

- CPU Clock:
La velocidad de la CPU emulada se puede ajustar entre 10% y 200%. El valor nominal es 100% y la CPU se emula con más exactitud.
Usa valores más bajos para ganar rendimiento.
También se pueden usar valores más altos hasta un 200% para evitar ralentizaciones en algunos juegos.

- CPU BIAS:
Auto: El multiplicador de ciclos de CPU se ajusta automáticamente.
1..16: El multiplicador de ciclos de CPU se ajusta manualmente.
3 es un valor bueno para la mayor parte de los juegos, pero el valor más óptimo depende de cada juego.
4 puede usarse en algunos juegos 2D para ganar velocidad.
Los valores más altos son más rápidos porque se hace underclock de la CPU, pero si el juego necesita mucha CPU el juego se ralentizará.
Los valores más bajos son más lentos pero pueden ser requeridos por algunos juegos.

- CPU Core:
HLE: El modo más rápido. El recompilador esta habilitado con emulación de BIOS.
HLE-Secure: Más lento pero más compatible (ejemplo: Castlevania SOTN ó Final Fantasy VII lo necesitan para funcionar).
BIOS: Más lento que HLE pero más compatible. Es necesaria una copia de la BIOS de la PSX (scph1001.bin).
HLE (Interpreter): Se utiliza el intérprete (más lento que el recompilador) con emulación BIOS (HLE).
BIOS (Interpreter): Se utiliza el intérprete junto con una copia de la BIOS de la PSX (scph1001.bin).
NOTE: Los modos BIOS necesitan una copia de la BIOS de la PSX (scph1001.bin) colocada en el mismo directorio del emulador.
Es necesario para jugar a juegos como Cotton 100% ó Bubble Bobble 2 con sonido. También es necesario con otros juegos
debido a fallos en la emulación de la BIOS mediante HLE (tarjetas de memoria, temporización del sonido en PAL, etc.).

- GPU Type:
Software: GPU por software con todas sus características habilitadas.
No Light: Desactivar iluminación para ganar velocidad. Puede provocar errores gráficos en los juegos.
No Blend: Desactivar blending para ganar velocidad. Puede provocar errores gráficos en los juegos.
No Light+Blend: Desactivar iluminación y blending. Puede provocar errores gráficos en los juegos.

- Auto-Save:
OFF: El modo de auto-guardado está desactivado.
ON: El estado del juego se carga automáticamente al comenzar y se graba al salir.

- Game Fixes:
Parasite Eve 2, Vandal Hearts 1/2 fix
InuYasha Sengoku Battle fix


4. INSTALACIÓN
--------------

autorun.gpu         -> Script de auto-run
mcd001.mcr          -> Memory-card #1
mcd002.mcr          -> Memory-card #2
pcsx4all.gpe        -> Script de selección automática WIZ/CAANOO
pcsx4all_wiz.gpe    -> Menú de selección de juegos (WIZ)
pcsx4all_wiz        -> Ejecutable del emulador (WIZ)
pcsx4all_caanoo.gpe -> Menú de selección de juegos (CAANOO)
pcsx4all_caanoo     -> Ejecutable del emulador (CAANOO)
scph1001.bin        -> Copia de la BIOS de la PSX (no incluida)
warm_2.6.24.ko      -> MMU Hack Kernel Module
conf/               -> Ficheros de configuración
exec/               -> Directorio de ejecutables PSX
isos/               -> Directorio de ISOs PSX
save/               -> Directorio para los save-states


5. SOPORTE DE JUEGOS
--------------------

- Los siguientes formatos de ISOs están soportados: iso, bin, bin+cue, bin+toc, img+ccd, mdf+mds, img+sub.
- ¡Las extensiones de las ISOs deben de estar en minúsculas!
- No borres los ficheros cue, toc, ccd, mds o sub, son necesarios para el soporte de CD-Audio.
- ¡Los formatos ISO comprimidos todavía no están soportados!
- Adicionalmente los ejecutables PSX también están soportados.

15. COMPILACIÓN
---------------

Compilar ejecutable para WIZ: DevKitGP2X rc2 (http://sourceforge.net/project/showfiles.php?group_id=114505)
make -f Makefile.wiz clean
make -f Makefile.wiz

NOTA: Para compilar ejecutable para WIZ es necesario modificar los siguientes dos ficheros de DevKitGP2X manualmente:
devkitGP2X\sysroot\usr\lib\libm.so:
/* GNU ld script
   Use the shared library, but some functions are only in
   the static library, so try that secondarily.  */
OUTPUT_FORMAT(elf32-littlearm)
GROUP ( /lib/libm.so.6 )
devkitGP2X\sysroot\usr\lib\libpthread.so:
/* GNU ld script
   Use the shared library, but some functions are only in
   the static library, so try that secondarily.  */
OUTPUT_FORMAT(elf32-littlearm)
GROUP ( /lib/libpthread.so.0 )

Compilar ejecutable para CAANOO: Sourcery G++ Lite 2006q1-6 (http://www.codesourcery.com/sgpp/lite/arm/portal/release293)
make -f Makefile.caanoo clean
make -f Makefile.caanoo
